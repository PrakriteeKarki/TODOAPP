#Todo Application

A simple ToDo Application built using django.

Thsi project allows users to :
-Add tasks 
-Edit taks
-Delete tasks
-Mark tasks as completed

The frontencd is build using HTML,Bootstrap,and Django Templates .

---

##Features 

- create new tasks
- -Update existing tasks
- -Delete tasks
- Mark task as done and undone
- User-friendly interface
- Django template rendering
- SQLite database integration

---

### Technologies Used

-Python 
-Django
-HTML
-Bootstrap
-SQLite
